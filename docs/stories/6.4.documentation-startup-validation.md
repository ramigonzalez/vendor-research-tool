   - "LLM Provider" subsection with env var table (`LLM_PROVIDER`, `LLM_MODEL`, `LLM_TEMPERATURE`, API keys)
   - Quick-switch `.env` snippets for each provider (OpenRouter default, Anthropic, OpenAI)
   - Multi-key coexistence explanation
2. `README.md` Prerequisites section updated: Anthropic no longer required, OpenRouter listed as default, all three providers documented with signup links.
3. `README.md` Tech Stack table updated: LLM row shows "Configurable: OpenRouter (default), Anthropic, OpenAI".
4. `run.sh` API key validation updated to check only the key for the selected `LLM_PROVIDER` using a `case` statement.
5. Startup log line in `app/main.py` prints `"LLM: {provider} / {model}"` on boot via `logger.info()`. Never prints API keys.
