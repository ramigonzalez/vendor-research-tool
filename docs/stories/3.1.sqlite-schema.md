## Story
**As a** persistence layer,
**I want** a SQLite database with jobs, evidence, and scores tables initialized on app startup,
**so that** all research results are persisted durably and can be retrieved after the pipeline completes.

## Acceptance Criteria
1. `jobs` table: `id TEXT PRIMARY KEY`, `status TEXT NOT NULL`, `created_at TIMESTAMP NOT NULL`, `completed_at TIMESTAMP`, `progress_pct INTEGER DEFAULT 0`, `progress_message TEXT DEFAULT ''`, `summary TEXT`, `rankings_json TEXT`.
2. `evidence` table: `id INTEGER PRIMARY KEY AUTOINCREMENT`, `job_id TEXT NOT NULL REFERENCES jobs(id)`, `vendor TEXT NOT NULL`, `requirement_id TEXT NOT NULL`, `claim TEXT`, `source_url TEXT`, `source_name TEXT`, `source_type TEXT`, `content_date TEXT`, `relevance REAL`, `supports_requirement INTEGER`, `created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP`.
3. `scores` table: `id INTEGER PRIMARY KEY AUTOINCREMENT`, `job_id TEXT NOT NULL REFERENCES jobs(id)`, `vendor TEXT NOT NULL`, `requirement_id TEXT NOT NULL`, `score REAL`, `confidence REAL`, `capability_level TEXT`, `maturity TEXT`, `justification TEXT`, `limitations_json TEXT`, `UNIQUE(job_id, vendor, requirement_id)`.
4. Index on `evidence(job_id)` and `scores(job_id)`.
5. WAL mode enabled: `PRAGMA journal_mode=WAL`.
6. `create_db_and_tables()` async function in `app/repository.py` called during FastAPI lifespan startup.
7. Database file: `research.db` in project root. Added to `.gitignore`.
8. No SQLite-specific syntax in DML queries (use standard SQL for PostgreSQL compatibility).

## Dev Notes
**Depends on**: Story 2.1 (models) — need `JobStatus`, `SourceType` enums for understanding field constraints.

**SQLite database path**: Use `pathlib.Path(__file__).parent.parent / "research.db"` for relative path resolution.

**aiosqlite pattern**:
```python
import aiosqlite
import asyncio

DB_PATH = "research.db"

async def create_db_and_tables():
    async with aiosqlite.connect(DB_PATH) as db:
        await db.execute("PRAGMA journal_mode=WAL")
        await db.execute("""
            CREATE TABLE IF NOT EXISTS jobs (
                id TEXT PRIMARY KEY,
                status TEXT NOT NULL,
                ...
            )
        """)
        await db.commit()
```

**FastAPI lifespan**:
```python
from contextlib import asynccontextmanager

@asynccontextmanager
async def lifespan(app: FastAPI):
    await create_db_and_tables()
    yield

app = FastAPI(lifespan=lifespan)
```

**`supports_requirement`** stored as INTEGER (0/1) in SQLite — convert to bool on read.

**`limitations_json`** stored as JSON string — serialize with `json.dumps()`, deserialize with `json.loads()`.
