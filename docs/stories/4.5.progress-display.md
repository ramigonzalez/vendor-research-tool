## Story
**As a** user triggering research,
**I want** to see a live progress bar and status message while the pipeline runs,
**so that** I know the system is working and what phase it's in.

## Acceptance Criteria
1. "Run Research" button triggers POST to `/api/research` and opens SSE stream.
2. Progress bar fills 0→100% based on `pct` field from SSE progress events.
3. Status message below bar shows `message` from SSE event (e.g., "Researching: LangSmith × Tracing...").
4. Phase label shown (Research / Scoring / Synthesis).
5. "Run Research" button disabled while pipeline is running.
6. On `completed` event: progress section hides, results section renders.
7. On `error` event: error message displayed with a "Retry" button.
8. Progress bar width transition is CSS animated (smooth fill).

**SSE consumption** (from planning docs):
```javascript
async function runResearch() {
  document.getElementById('run-btn').disabled = true;
  document.getElementById('progress-section').classList.remove('hidden');

  const response = await fetch('/api/research', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({})
  });

  const reader = response.body.getReader();
  const decoder = new TextDecoder();

  while (true) {
    const {done, value} = await reader.read();
    if (done) break;

    const chunk = decoder.decode(value);
    const lines = chunk.split('\n');
    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const event = JSON.parse(line.slice(6));
        handleSSEEvent(event);
      }
    }
  }
}

function handleSSEEvent(event) {
  if (event.type === 'progress') {
    document.getElementById('progress-bar').style.width = event.pct + '%';
    document.getElementById('progress-message').textContent = event.message;
  } else if (event.type === 'completed') {
    document.getElementById('progress-section').classList.add('hidden');
    renderResults(event.results);
  } else if (event.type === 'error') {
    showError(event.message);
  }
}
```
