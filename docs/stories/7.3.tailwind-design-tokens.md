## Story
**As a** developer,
**I want** Tailwind CSS configured with our design token system,
**so that** all components use consistent colors, typography, spacing, and animations from a single source of truth.

## Acceptance Criteria
1. [x] All `docs/ux/tokens.yaml` values mapped to Tailwind v4 `@theme` block in `globals.css`.
2. [x] Colors: light/dark mode with all status, confidence, accent colors.
3. [x] Typography: Inter (UI), Source Serif 4 (AI content), JetBrains Mono (code) with `font-display: swap`.
4. [x] Custom keyframe animations: shimmer, token-fade-in, step-pulse, pill-enter.
5. [x] `@media (prefers-reduced-motion: reduce)` resets all animations.
6. [x] Dark mode support via `@media (prefers-color-scheme: dark)`.

## Tasks / Subtasks
- [x] Map color tokens to CSS custom properties in `@theme` block
