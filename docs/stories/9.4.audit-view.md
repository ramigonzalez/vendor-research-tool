## Story
**As a** user,
**I want** a tabbed interface to browse different aspects of the research audit trail,
**so that** I can explore queries, sources, and activity in an organized manner.

## Acceptance Criteria
1. [x] Tabs: Timeline | Queries | Sources | Activity Log.
2. [x] Accessible: `role="tablist"` / `role="tab"` / `role="tabpanel"`, arrow-key navigation.
3. [x] Populated from `useResearchState` hook data.
4. [x] "Audit trail not available for this run" message for historical jobs without data.
5. [x] Tab counts update in real-time (e.g., "Queries (48)", "Sources (24)").
