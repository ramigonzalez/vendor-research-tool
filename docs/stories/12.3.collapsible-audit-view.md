# Story 12.3: Collapsible Research Analysis

## Description
The Research Analysis section takes too much vertical space after completion. Add a disclosure card pattern that collapses the section by default when research is complete.

## Acceptance Criteria
- [x] Collapsible card wraps the existing tab interface
- [x] Header shows "Research Analysis" with source/query counts and chevron
- [x] During active research: always expanded, header non-interactive
- [x] After completion: starts collapsed, toggleable with smooth animation
- [x] Existing tab bar and panels unchanged inside collapsible content
