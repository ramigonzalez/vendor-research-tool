## Story
**As a** frontend,
**I want** to POST to `/api/research` and receive real-time pipeline progress via Server-Sent Events,
**so that** research runs asynchronously and the UI updates live without polling.

## Acceptance Criteria
1. `POST /api/research` accepts optional JSON body: `{"vendors": [...], "requirements": [...]}` (uses config defaults if omitted).
2. Creates a new `ResearchJob` in SQLite with status `running`, returns `StreamingResponse`.
3. `Content-Type: text/event-stream` header on response.
4. Launches pipeline as an asyncio task while streaming events from progress queue.
5. SSE events streamed in order: `started` → `progress` (multiple) → `completed` or `error`.
6. Each event formatted: `data: {json_string}\n\n` (SSE spec compliant).
7. On pipeline success: emits `{"type": "completed", "results": <ResearchResults as dict>}`.
8. On pipeline failure: emits `{"type": "error", "message": "<error_message>"}`, updates job status to `failed`.
9. CORS configured to allow `http://localhost:8000` and `http://127.0.0.1:8000`.
10. Job status updated to `completed` or `failed` in SQLite on stream end.

**SSE endpoint pattern**:
```python
from fastapi.responses import StreamingResponse
import asyncio, json, uuid

@router.post("/api/research")
async def run_research(
    repo: ResearchRepository = Depends(get_repository)
):
    job_id = str(uuid.uuid4())
    job = ResearchJob(id=job_id, status=JobStatus.running, created_at=datetime.now())
    await repo.create_job(job)

    queue: asyncio.Queue = asyncio.Queue()

    async def event_generator():
        yield f"data: {json.dumps({'type': 'started', 'job_id': job_id})}\n\n"

        state = build_initial_state(job_id, queue)
        pipeline_task = asyncio.create_task(run_pipeline(state, repo))

        while not pipeline_task.done() or not queue.empty():
            try:
                event = await asyncio.wait_for(queue.get(), timeout=0.5)
                yield f"data: {json.dumps(event)}\n\n"
            except asyncio.TimeoutError:
                continue

        if pipeline_task.exception():
            err = str(pipeline_task.exception())
            yield f"data: {json.dumps({'type': 'error', 'message': err})}\n\n"
            await repo.update_job_status(job_id, JobStatus.failed, 100, err)
        else:
            results = await repo.get_results(job_id)
            yield f"data: {json.dumps({'type': 'completed', 'results': results.model_dump(mode='json')})}\n\n"

    return StreamingResponse(event_generator(), media_type="text/event-stream")
```

**`model_dump(mode='json')`**: ensures datetimes serialized to ISO strings by Pydantic v2.
