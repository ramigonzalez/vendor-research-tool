## Story
**As a** user running a research job,
**I want** to see which stage the research pipeline is in,
**so that** I understand what's happening and how far along the process is.

## Acceptance Criteria
1. [x] Horizontal stepper: Planning -> Searching -> Analyzing -> Scoring -> Ranking -> Complete.
2. [x] Step states: pending (gray), active (animated pulse), complete (green check).
3. [x] Connecting line fills progressively with transition animation.
4. [x] Active step colors from design tokens.
5. [x] Elapsed time counter (MM:SS) displayed next to stepper.
6. [x] Responds to `phase_start`/`phase_end` SSE events via `useResearchState` hook.
7. [x] Responsive: mobile (<md) condenses to current phase icon + label + time.
