## Story
**As a** user with accessibility needs,
**I want** the UI to be navigable via keyboard and screen reader,
**so that** I can use the research tool regardless of how I interact with the interface.

## Acceptance Criteria
1. [x] ScoreCell: `role="button"`, `tabIndex={0}`, `aria-label` with vendor+requirement+score.
2. [x] DrillDownPanel: `role="dialog"`, `aria-labelledby`, focus trap (useEffect), return-focus on close.
3. [x] Escape key closes the drill-down panel.
4. [x] Skip-to-content link at top of page.
5. [x] Visible focus rings: `focus-visible:ring-2 focus-visible:ring-accent-primary`.
