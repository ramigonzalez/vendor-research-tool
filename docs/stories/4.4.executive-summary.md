## Story
**As a** researcher/evaluator,
**I want** an AI-generated executive summary displayed above the comparison matrix,
**so that** I get a narrative overview and top recommendation before diving into scores.

## Acceptance Criteria
1. Executive summary section displayed above the comparison matrix.
2. Summary rendered as formatted prose (3–4 paragraphs, preserving line breaks).
3. "Executive Summary" heading with clear visual separation from matrix.
4. Rankings visualization displayed below summary: ordered list showing vendor rank, name, overall score (e.g., "#1 LangSmith — 8.4/10").
5. Summary section has max-width ~750px for readability.
6. Executive summary generated by LLM as LangGraph `synthesize` node — see Dev Notes.
7. Fallback text shown if `results.summary` is empty: "Executive summary not available — see matrix for detailed scores."

**Summary generation prompt** (from planning docs):
```
You are writing an executive summary for an LLM observability platform evaluation.

Evaluation Results:
- Vendors evaluated: {vendors}
- Requirements: {requirements_with_priorities}
- Rankings: {vendor_rankings_with_scores}
- Top scores by requirement: {top_scores_per_requirement}
- Notable gaps (low confidence): {low_confidence_pairs}

Write a 3-4 paragraph executive summary. Include:
1. Overall recommendation (which vendor and why)
2. Key differentiators between top 2 vendors
3. Notable strengths/weaknesses per vendor
4. Confidence gaps that warrant further investigation

Write in prose (no bullet points). Be specific and cite evidence-based scores.
Max 400 words.
```

**LangGraph pipeline node order**: `compute_rankings → generate_summary → [end]`

**Frontend rendering**:
```javascript
function renderSummary(results) {
  const summaryEl = document.getElementById('executive-summary');
  if (!results.summary) {
    summaryEl.innerHTML = '<p class="no-summary">Executive summary not available...</p>';
    return;
  }
  // Split by double newline into paragraphs
  const paragraphs = results.summary.split('\n\n').filter(p => p.trim());
  summaryEl.innerHTML = paragraphs.map(p => `<p>${p}</p>`).join('');

  // Render rankings
  const rankingsEl = document.getElementById('rankings-list');
  rankingsEl.innerHTML = results.rankings
    .map(r => `<li><strong>#${r.rank} ${r.vendor}</strong> — ${r.overall_score.toFixed(1)}/10</li>`)
    .join('');
}
```
