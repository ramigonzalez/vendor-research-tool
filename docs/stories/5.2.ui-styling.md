## Story
**As a** evaluator reviewing the tool,
**I want** a clean, professional visual design,
**so that** the tool looks production-quality and data is easy to read.

## Acceptance Criteria
1. CSS custom properties defined: `--color-high: #2ecc71`, `--color-medium: #f39c12`, `--color-low: #e74c3c`, `--color-bg: #f8f9fa`, `--color-surface: #ffffff`, `--color-text: #2c3e50`.
2. Page layout: centered content max-width 1200px, responsive padding.
3. Matrix table: sticky vendor header row on scroll, alternating row backgrounds.
4. Vendor header cells show overall rank badge (e.g., "#1" pill badge).
5. Typography: system font stack, 14px for matrix cells, 16px body, 24px headings.
6. Responsive: matrix scrollable horizontally on < 768px screens.
7. "Could Have" flag: may be cut if time-constrained. Must not block MVP.
