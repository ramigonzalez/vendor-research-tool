## Story
**As a** user running a research job,
**I want** to see sources appearing in real-time with animations similar to Perplexity/ChatGPT deep search,
**so that** the research process feels dynamic and I can see progress being made.

## Acceptance Criteria
1. [x] Vertical timeline below stepper, one row per phase.
2. [x] Active row: animated pulse on icon.
3. [x] SourcePill molecules animate in as `search_result` events arrive.
4. [x] SourcePill: domain text, rounded-full pill, entrance animation (fade + translateY + scale).
5. [x] Hover expands to show full title.
6. [x] Live source count updates in real-time.
7. [x] Searching sub-phases: shows iteration info from `iteration_start` events.
