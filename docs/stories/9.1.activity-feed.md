## Story
**As a** user,
**I want** to see a live log of what the research agent is doing,
**so that** I can understand each step of the process in real-time and catch any warnings or errors.

## Acceptance Criteria
1. [x] Scrollable feed with relative timestamps (+0:34).
2. [x] Maps SSE event types to human-readable messages (query_generated, search_result, evidence_extracted, score_computed, warning, iteration_start).
3. [x] Warning entries styled with amber highlight (`bg-amber-50 border-l-2 border-amber-400`).
4. [x] New entries at top with fade-in animation.
5. [x] Max 200 entries in state.
6. [x] Auto-scroll unless user scrolled up.
