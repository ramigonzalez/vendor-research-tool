## Story
**As a** evaluator or new developer,
**I want** a comprehensive README,
**so that** I can set up and run the tool without reading source code.

## Acceptance Criteria
1. `README.md` includes: project description, quick start (clone → .env → install → run).
2. Prerequisites section: Python 3.11+, API keys (Anthropic + Tavily) with signup links.
3. Architecture section: pipeline description (3 phases), scoring methodology summary.
4. Scoring transparency section: explains the 4-factor score + 4-factor confidence formulas.
5. Configuration section: how to modify VENDORS, REQUIREMENTS, weights in `app/config.py`.
6. Scale section: how architecture would scale from 4 vendors to 50+ (Celery, Redis, vector store).
7. Known limitations: fixed vendors, local-only, ~3 min runtime.
8. Setup instructions tested against a clean Python 3.11 virtualenv.

## Dev Notes

```markdown
## Quick Start
git clone <repo>
cd vendor-research-tool
cp .env.example .env
# Edit .env with your API keys
pip install -r requirements.txt
uvicorn app.main:app --reload
# Open http://localhost:8000
```

**API Key links**: Anthropic → console.anthropic.com, Tavily → app.tavily.com

**Runtime estimate**: ~3 minutes for full 4×6 research run (55–60 Tavily searches + LLM calls).
