## Story
**As a** user viewing a vendor-requirement drill-down,
**I want** to see exactly which factors contributed to the score,
**so that** I can understand why a vendor scored high or low for a specific requirement.

## Acceptance Criteria
1. [x] "Score Breakdown" section in DrillDownPanel below overall score.
2. [x] Shows 4 components: Capability level -> mapped score, Evidence strength, Maturity level -> mapped score, Limitations count -> penalty.
3. [x] Visual formula: `0.40 x 10.0 + 0.30 x 6.0 + 0.20 x 7.0 + 0.10 x 8.0 = 8.0`.
4. [x] Color coding: green (contributing positively), red (pulling score down).
5. [x] Lookup tables match `engine.py` (CAPABILITY_SCORES, MATURITY_SCORES).
