# Story 12.1: Dark/Light/System Theme Toggle

## Description
Add a manual theme toggle that supports light, dark, and system (OS preference) modes. Currently the app relies solely on `@media (prefers-color-scheme: dark)` with no user override.

## Acceptance Criteria
- [x] `globals.css` uses `.dark` class instead of `@media (prefers-color-scheme: dark)`
- [x] `ThemeContext` provides `theme` state and `setTheme` function
- [x] Three modes: `light`, `dark`, `system`
- [x] Persists preference to `localStorage` key `vrt-theme-preference`
- [x] `system` mode listens to OS changes in real time
- [x] `ThemeToggle` atom with segmented 3-button control
- [x] `App.tsx` wraps with `<ThemeProvider>`
- [x] `index.html` inline script prevents flash of wrong theme

## Files
- `src/styles/globals.css` — `@media` → `.dark` class
- `src/contexts/ThemeContext.tsx` — NEW
- `src/components/atoms/ThemeToggle.tsx` — NEW
- `src/App.tsx` — Wrap with ThemeProvider
- `index.html` — Flash prevention script
