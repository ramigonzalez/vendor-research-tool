## Story
**As a** researcher/evaluator,
**I want** to click a matrix cell and see the evidence behind the score,
**so that** I can verify the AI's reasoning and check source quality myself.

## Acceptance Criteria
1. Clicking any matrix cell opens a slide-in panel on the right side (or modal overlay).
2. Panel header: vendor name, requirement name, score value, confidence value.
3. Panel body section 1: LLM justification paragraph.
4. Panel body section 2: limitations list (if any).
5. Panel body section 3: evidence items sorted by relevance descending. Each item shows:
   - Claim text (truncated to 300 chars with "..." if longer)
   - Source type badge (color-coded: official_docs=blue, github=dark, comparison=purple, blog=orange, community=grey)
   - Source name + clickable URL
   - Content date (if available) + relevance score
   - Supports requirement: ✓ or ✗ indicator
6. Panel closable via ✕ button and by clicking outside/overlay.
7. Panel opens within 100ms of click (no network request — data already in memory).
8. Multiple cells clickable: last click replaces previous panel content.

## Dev Notes
**Depends on**: Story 4.1 (matrix rendered with click handlers), Story 4.2 (evidence data in memory from results object).

**Panel HTML structure**:
```html
<div id="drill-down-panel" class="panel hidden">
  <div class="panel-overlay" onclick="closePanel()"></div>
  <div class="panel-content">
    <button class="panel-close" onclick="closePanel()">✕</button>
    <div id="panel-header"></div>
    <div id="panel-justification"></div>
    <div id="panel-limitations"></div>
    <div id="panel-evidence-list"></div>
  </div>
</div>
```

**Source type badge colors**:
```javascript
const BADGE_COLORS = {
  official_docs: '#3498db',  // blue
  github: '#2c3e50',         // dark
  comparison: '#9b59b6',     // purple
  blog: '#e67e22',           // orange
  community: '#95a5a6'       // grey
};
```

**Evidence data available in-memory**: stored in `window.currentResults.matrix[vendor][reqId].evidence` after first load — no additional network request needed.

**CSS panel animation**:
```css
.panel-content {
  position: fixed; right: 0; top: 0; bottom: 0;
  width: 420px; background: white;
  transform: translateX(100%);
  transition: transform 0.2s ease;
  overflow-y: auto; padding: 20px;
}
.panel.open .panel-content { transform: translateX(0); }
```
