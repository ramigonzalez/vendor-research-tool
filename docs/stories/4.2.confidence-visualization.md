## Story
**As a** researcher/evaluator,
**I want** each matrix cell to visually indicate evidence confidence,
**so that** I immediately see which scores are strongly backed vs. based on sparse evidence.

## Acceptance Criteria
1. High confidence (≥0.7): solid border, full opacity — no special indicator.
2. Medium confidence (0.4–0.69): dashed border (`border: 2px dashed #aaa`), 80% opacity.
3. Low confidence (<0.4): dashed red border (`border: 2px dashed #e74c3c`), 60% opacity, ⚠️ icon in cell.
4. Confidence value shown as small subscript text in cell: "conf: 0.85".
5. Legend displayed below matrix explaining confidence indicators.
6. Confidence display integrates seamlessly with score color from Story 4.1.

