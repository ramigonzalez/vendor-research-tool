## Story
**As a** scoring engine,
**I want** to compute a 0–1 confidence score from evidence metadata using a fully deterministic algorithm,
**so that** confidence accurately reflects evidence quality without any LLM opinion.

## Acceptance Criteria
1. `compute_confidence(evidence: list[Evidence]) -> float` in `app/scoring/engine.py`.
2. Source Count (30%): `min(len(evidence), 5) / 5` — 5+ sources = full score.
3. Source Authority (30%): mean authority weight per source: `official_docs=1.0, github=0.8, comparison=0.6, blog=0.4, community=0.3`.
4. Source Recency (25%): proportion scoring: within 6 months = 1.0, 7–12 months = 0.7, older or date unknown = 0.3.
5. Consistency (15%): if all evidence agrees on `supports_requirement` → 1.0; if mixed → proportion of `True` values; if all `False` → 0.0.
6. Final confidence = weighted sum, clamped to [0.0, 1.0].
7. Empty evidence list → returns `0.0`.
8. Function is pure — no side effects, no LLM calls.
9. Result used in `ScoreResult.confidence` and displayed in UI.

**Confidence computation** (verbatim from planning docs):
```python
from datetime import datetime, timedelta
from dateutil import parser as date_parser  # pip install python-dateutil

AUTHORITY_WEIGHTS = {
    SourceType.official_docs: 1.0,
    SourceType.github: 0.8,
    SourceType.comparison: 0.6,
    SourceType.blog: 0.4,
    SourceType.community: 0.3,
}

def compute_confidence(evidence: list[Evidence]) -> float:
    if not evidence:
        return 0.0

    # Source Count (30%)
    count_score = min(len(evidence), 5) / 5

    # Source Authority (30%)
    authority_scores = [AUTHORITY_WEIGHTS.get(e.source_type, 0.3) for e in evidence]
    authority_score = sum(authority_scores) / len(authority_scores)

    # Source Recency (25%)
    now = datetime.now()
    recency_scores = []
    for e in evidence:
        if not e.content_date:
            recency_scores.append(0.3)
            continue
        try:
            date = date_parser.parse(e.content_date)
            age_days = (now - date).days
            if age_days <= 180:
                recency_scores.append(1.0)
            elif age_days <= 365:
                recency_scores.append(0.7)
            else:
                recency_scores.append(0.3)
        except (ValueError, OverflowError):
            recency_scores.append(0.3)
    recency_score = sum(recency_scores) / len(recency_scores)

    # Consistency (15%)
    supporting = sum(1 for e in evidence if e.supports_requirement)
    consistency_score = supporting / len(evidence)

    confidence = (
        0.30 * count_score +
        0.30 * authority_score +
        0.25 * recency_score +
        0.15 * consistency_score
    )
    return max(0.0, min(1.0, confidence))
```

**`python-dateutil`** must be in requirements.txt for flexible date parsing.

**Edge cases**: `content_date` may be `None`, empty string, or various formats ("2024-03-15", "March 2024", "2024"). Use try/except.
